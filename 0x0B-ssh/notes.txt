server - a piece of computer hardware that provides functionality for other programs or devices, called clients. A singled server can servve multiple clients
A single server can serve multiple clients, and a single client can use multiple servers. A client process may run on the same device or may connect over a network to
a server on a different device.[1] Typical servers are database servers, file servers, mail servers, print servers, web servers, game servers, and application servers.
